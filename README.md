# bruCalculo

App web moderno e responsivo para controle de lucro de produtos.

- Visual feminino, delicado e mobile first
- Dashboard de lucros
- Histórico de produtos
- Gráfico de lucros
- Feito com HTML, CSS, JS puro

## Como publicar no GitHub Pages

1. No GitHub, vá em Settings > Pages
2. Em "Source", selecione o branch `main` e a pasta `/root` (ou `/`)
3. Salve. O site estará disponível em breve no link gerado pelo GitHub Pages.

Ou publique facilmente no Netlify arrastando a pasta para o painel do Netlify.
